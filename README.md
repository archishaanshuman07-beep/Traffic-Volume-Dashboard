# Traffic-Volume-Dashboard

A Streamlit dashboard for exploring and visualizing traffic volume patterns using the Metro Interstate Traffic Volume dataset.

## Features
- Displays the first few rows of the dataset for quick inspection.
- Compares daytime and nighttime traffic volume distributions using histograms.
- Visualizes average monthly traffic volume with a line chart.
- Explores the relationship between temperature and traffic volume using a - color-coded scatter plot by day of the week.
- Built with Streamlit and Altair 

## Database
The project uses the Metro Interstate Traffic Volume dataset found here: https://archive.ics.uci.edu/dataset/492/metro+interstate+traffic+volume
